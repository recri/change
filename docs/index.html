Change

This is a github.io page for Change, a progressive web app for consulting the Book of Changes.

The license for this project is LICENSE.txt.

The software is derived from the PWA Starter Kit and depends heavily on Lit Html and Lit Element from Google's Polymer Project. (And, no, I didn't forget to use the MWC Material Web Components, but they really aren't ready at the moment.)

The content is derived from many versions of the Book of Changes on the web and may be subject to additional copyrights.
